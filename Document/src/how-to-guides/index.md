# 🛠️ How-to Guides（ハウツーガイド）

*問題解決指向 - 特定のタスクを達成する方法*

How-to Guidesは、NijoApplicationBuilderを使って業務システムの外部仕様設計における特定のタスクを達成するための実践的な手順を示します。実際の外部仕様設計で遭遇する具体的な問題を解決する方法を学べます。
